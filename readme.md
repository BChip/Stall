# Description
The "Stall" GitHub repository hosts a social media Chrome extension that enables users to rate and chat on every webpage. The extension's installation involves a straightforward process of loading the unpacked extension into Chrome after building it with provided commands. Additionally, the repository details how to work with Firebase Functions, including local testing and deployment. # Extension

```
cd extension
yarn
yarn dev
```

- Open Chrome
- Go to: chrome://extensions/
- Turn on developer mode
- Click Load unpacked
- Select extension/build/chrome-mv3-dev folder

# Firebase Function

Install firebase CLI

`firebase login` log into firebase cli
`yarn serve` run locally
`yarn deploy` deploy
